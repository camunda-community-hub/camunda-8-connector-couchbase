# Code of Conduct

This project adheres to the [Camunda Community Code of Conduct](https://camunda.com/events/code-conduct/) and the [Camunda Community Values](https://camunda.com/developers/community-values/).

All contributors, maintainers, and participants are expected to follow these guidelines in all project spaces — GitHub issues, pull requests, discussions, and any other community forums.

## Summary

- Be kind, respectful, and constructive in all interactions.
- Harassment, discrimination, or personal attacks of any kind will not be tolerated.
- Focus on the technical merits of contributions and ideas.

## Reporting

If you experience or witness unacceptable behaviour, please report it by opening a [GitHub issue](../../issues/new) or contacting the maintainer directly.

For broader Camunda community concerns, refer to the [Camunda Code of Conduct reporting process](https://camunda.com/events/code-conduct/).
